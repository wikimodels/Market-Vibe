Справочник: Movement Efficiency (ER)

Суть метрики

Movement Efficiency — это КПД цены. Отношение "полезного" смещения к "затраченным усилиям".

В этой системе метрика направленная (от -1.0 до +1.0).

Формула расчёта

ER = (Priceend - Pricestart) / Σ|Pricei - Pricei-1|
# Пример: Net Move = +10 Total Move = 18 ER = 10 / 18 = 0.556 (Бычий тренд)

Интерпретация значений

Значение Состояние рынка Действие
> +0.40 Сильный бычий тренд Держать LONG. Не шортить!
-0.10 ... +0.10 Шум / Боковик Не торговать.
< -0.40 Сильный медвежий тренд Держать SHORT. Не ловить дно!

⚡ ДВА ЗОЛОТЫХ ПРАВИЛА

1. Правило Шорта (Ловля ножей): Пока ER < -0.40, закрывать шорт рано. Попытка купить дно=поймать нож.
2. Правило Лонга (Паровоз): Пока ER > +0.40, шортить = встать перед поездом. Ждите падения эффективности.

Почему RSI не работает? При высокой ER он будет показывать "Перекупленность" (>70) весь путь от $100 до $10,000.

Классификация (1D)

def classify(er_1d): if er_1d > 0.40: return "ELITE BULL TRENDY" if er_1d < -0.40: return "ELITE BEAR TRENDY" if abs(er_1d) < 0.15: return "FLAT CHOPPER"
