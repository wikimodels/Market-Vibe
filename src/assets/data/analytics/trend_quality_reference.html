Trend Quality: Deep Analysis

1. Техническая Правда (Code Audit)

Метрика trend_quality_w20 показывает Среднюю Стабильность Тренда.

Алгоритм Linear Regression R²
Тип данных Log Prices (экспоненциальный рост = прямая)
Окно 20 свечей (Скользящее)
# Логика: 1. Логарифмируем цены 2. Считаем R² в окне 20 3. Возвращаем среднее "качество" за всю историю

2. Физика: R-Squared (R²)

R² = 1.0 → Идеальная Прямая Линия (Рельсы)
R² = 0.0 → Облако точек (Шум)

Метрика оценивает "Геометрическую Чистоту" движения.

  • Памп и Дамп: Качество низкое (взлет и падение ломают линейность).
  • Органический Рост: Качество высокое (0.8-0.9).

3. Декодер Значений

MESSY (< 0.3) NORMAL (~0.5) RAILROAD (> 0.8)
TQ Score Характер Стратегия
Высокий (> 0.7) "Послушная" Пробои. Цена идет линейно. Стопы в безопасности.
Средний (0.4 - 0.6) "Шумная" Стандартный рынок. Нужны подтверждения.
Низкий (< 0.3) "Рваная" Избегать трендов. Только Mean Reversion или пампы.

Синергия с другими метриками

TQ > 0.8 + MCI > 0.4 Идеальный, предсказуемый тренд
TQ > 0.8 + MCI < -0.2 Чистый откат в стабильном диапазоне
TQ < 0.3 + Jagginess> 0.6 Абсолютный хаос

TQ-скрининг

Высокая TQ = зелёный свет для тренд-фолловинга. Низкая TQ = сигнал к отказу от направленных стратегий.

Эта метрика — ваш калибр рельсов. Торгуйте только там, где цена уважает линейность.
